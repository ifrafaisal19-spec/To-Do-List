# To-Do List (HTML/CSS/JavaScript)

A terminal-styled to-do list web app. Vanilla HTML, CSS, and JavaScript, no
frameworks, no build step, just open index.html in a browser.

## Features

- Add, complete, and delete tasks
- Filter by All / Active / Done
- "Clear done" to bulk-remove completed tasks
- Fully keyboard accessible, responsive down to mobile


## Project structure

index.html
style.css
script.js
README.md
